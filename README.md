# BuyZone - Your Ultimate Online Shopping Experience

## Project Overview
Welcome to **BuyZone**, an innovative online shopping platform designed to enhance your retail experience. BuyZone is built to provide customers with a seamless and enjoyable way to browse, select, and purchase products. BuyZone offers user-friendly design and admin control for a smooth experience.

## Key Features

### For Customers:
- **Product Browsing**: Discover a wide range of products with detailed descriptions and images.
- **Shopping Cart**: Add items to your cart for an easy checkout experience.
- **Wishlist**: Save products for later with a dedicated wishlist feature.
- **Secure Checkout**: Purchase your favorite items securely and conveniently.

### For Admins:
- **User Management**: View and manage user details with ease.
- **Order Tracking**: Monitor customer orders and track their status.
- **Inventory Management**: Keep the product catalog updated and organized.
- **Website Handling**: Maintain and update the website’s content to keep it engaging.

## Technology Stack
BuyZone is developed using:
- **Django**: A powerful Python web framework for the backend.
- **HTML, CSS, JavaScript**: For building responsive and interactive frontend pages.
- **MySQL**: As the database for storing user and product information.

## Installation and Setup
Follow these steps to set up the project locally:
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/BuyZone.git
   ```
2. **Navigate to the Project Directory**:
   ```bash
   cd BuyZone
   ```
3. **Create a Virtual Environment**:
   ```bash
   python -m venv env
   ```
4. **Activate the Virtual Environment**:
   - On Windows:
     ```bash
     .\env\Scripts\activate
     ```
   - On macOS/Linux:
     ```bash
     source env/bin/activate
     ```
5. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
6. **Run Migrations**:
   ```bash
   python manage.py migrate
   ```
7. **Start the Development Server**:
   ```bash
   python manage.py runserver
   ```
